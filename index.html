使用hexo来创建博客

2020-06-21


github图片路径：https://github.com/luckygoodman/img_folder/raw/master/4.jpg

第一章：

(仅在win10环境上，linux上命令有些许不同请自行百度)

环境安装：

Node.js下载（node,nmp -v查看是否安装上）

第二章：

（获得最高管理权限，win系统即管理员命令打开命令行）
工具安装：

安装淘宝镜像：npm install -g cnpm --registry=https://registry.npm.taobao.org

hexo脚手架: cnpm install -g hexo -cli

git部署git插件：cnpm install --save hexo-deployer-git

第三章：

hexo的yml文件配置：

在Deployment下配置：
type:  git
repo:  github仓库地址
branch：  master（可配）

第四章：

hexo部署命令：
初始化： hexo init

新建： hexo n

清理： hexo clean

项目部署： hexo g

推送git： hexo d

部署项目： hexo s
